# 🚀 AirSense

O AirSense é um poste inteligente que utiliza de sensores, display e ESP32 para informar, horário, temperatura e qualidade de ar

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** C++
* **Plataforma:** ESP32
* **Simulador:** [Wokwi](https://wokwi.com/)

---

## 💻 Como Rodar o Projeto

Você não precisa de nenhum componente físico para testar este projeto! Ele foi totalmente desenvolvido e testado no simulador online Wokwi. Escolha uma das opções abaixo para rodar:

### Opção 1: Rodar direto no Navegador (Mais Rápido ⚡)
1. Acesse o link do projeto já configurado no Wokwi: [https://wokwi.com/projects/463358063967261697]
2. Clique no botão de **Play (Iniciar Simulação)** no canto superior esquerdo da tela.
3. Interaja com os sensores na tela e veja o resultado aparecer no Display. Obs: O display demora para atualizar

### Opção 2: Rodar Localmente (VS Code + Extensão Wokwi)
Se você preferir rodar o projeto na sua máquina usando o VS Code, siga estes passos:

1. **Pré-requisitos:**
   * Ter o [VS Code](https://code.visualstudio.com/) instalado.
   * Instalar a extensão **Wokwi Simulator** no VS Code.
   * **Ative a licença do Wokwi (Gratuita):**
     * A extensão do Wokwi no VS Code precisa de uma licença ativa para funcionar.
     * Pressione `F1` (ou `Ctrl+Shift+P`), digite `Wokwi: Request a New License` e pressione Enter.
     * O VS Code vai abrir o seu navegador. Clique no botão **"GET YOUR LICENSE"** (você pode logar com sua conta do GitHub).
     * O navegador pedirá permissão para voltar ao VS Code. Aceite, e você verá uma mensagem confirmando que a licença foi ativada!
     *(Nota: Para projetos open-source a licença é grátis e dura 30 dias, podendo ser renovada pelo mesmo processo sempre que expirar).*
   * Instalar a extensão **PlatformIO** no VS code.

2. **Passo a Passo:**
   * **Se você tem o Git instalado:**
     * Crie uma pasta e abra ela no VScode
     * Abra o terminal e clone o repositório:
     ```bash
     git clone [https://github.com/seu-usuario/seu-repositorio.git](https://github.com/seu-usuario/seu-repositorio.git)
     ```
   * **Se você NÃO tem o Git instalado:**
     * Clique no botão verde **"Code"** (no topo desta página) e selecione **"Download ZIP"**. Depois, extraia os arquivos na sua máquina.
     * Abra a pasta do projeto no VS Code.
   * Para compilar o código: Clique no ícone de Checkmark (✓) na barra inferior do VS Code. O PlatformIO baixará todas as bibliotecas necessárias sozinho.
   * Pressione `F1` (ou `Ctrl+Shift+P` / `Cmd+Shift+P`) e digite: **Wokwi: Start Simulator**.
   * A simulação abrirá diretamente em uma aba do seu VS Code.
